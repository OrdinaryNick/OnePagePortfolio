# One Page Portfolio
Simple one html page as photo portfolio.

# How to run
1. Build docker image
```docker build -t ordinarynick/one-page-portfolio .```
2. Run docker image
```docker run -p 80:80 -d ordinarynick/one-page-portfolio```
